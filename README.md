# Python-Projeto-Automação 
 A ideia foi  criar uma automação de WhatsApp com Python para encaminhar uma ou demais mensagens para vários contatos.  Só que um dos requisitos é enviar essa mensagem de 5 em 5 contatos para que o WhatsApp não bloqueie essa conta, pois ele não aceita automações.  Como é um projeto para simular uma pessoa de fato fazendo essa ação, foi utilizado o Selenium no Python (para utilizar o navegador de forma automática), a biblioteca pyperclip (para copiar e colar informações) e o webdriver-manager para facilitar a utilização do selenium com o navegador (chrome)
